What is Python?

Python is a popular, easy programming language for scripts, games, data, web, AI, etc.

Basic Python Example

print("Hello, Python world!")

Variables and Math

a = 10 b = 5 print("Sum:", a + b) print("Product:", a * b)

If-Else (Conditions)

score = 56 if score >= 50: print("Pass") else: print("Fail")

Loops

for i in range(1, 6): print(i)

Functions

def greet(name): print("Hello,", name) greet("webTune1")

Mini Project: Simple Calculator

num1 = 7 num2 = 3 print("Sum:", num1 + num2) print("Difference:", num1 - num2) print("Product:", num1 * num2) print("Quotient:", num1 / num2)
Try: Change numbers and see the result.

Practice Questions

  • Write a program to print your name and your favorite food.
  • Print the numbers from 10 to 1 in reverse using a loop.

How to run Python:

  • Install Python: python.org
  • Write code in Notepad/VS Code. Save as test.py
  • Open Command Prompt, type: python test.py
