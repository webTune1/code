What is CSS?

CSS (Cascading Style Sheets) is the language for styling HTML pages: colors, spacing, layout, fonts, animations, and more.

CSS Syntax

selector {
property: value;
}
Example:
p { color: blue; font-size: 20px; }

Selectors

element #id .class * div p div > p a:hover
/* All p tags */
p { color: #333; }

/* Element with id="main" */
#main { background: #eaf2ff; }

/* All elements with class="btn" */
.btn { padding: 10px; }
  • #id for unique elements, .class for groups.
  • :hover for mouse-over effects.

Color & Background

color background background-color background-image opacity
body {
color: #222;
background: #f0f8ff;
}

h1 {
color: #0984e3;
}

div {
background-image: url("bg.jpg");
opacity: 0.8;
}
Use color names, hex codes (#0984e3), rgb(), or hsl().

Fonts & Text

font-family font-size font-weight color text-align text-decoration line-height letter-spacing text-shadow
h1 {
font-family: 'Figtree', Arial, sans-serif;
font-size: 2.2rem;
font-weight: 800;
text-align: center;
color: #0984e3;
text-shadow: 2px 2px 8px #eaf2ff;
}
p {
line-height: 1.6;
letter-spacing: 0.5px;
}

Box Model (Spacing & Borders)

margin padding border box-sizing width height border-radius
.box {
background: #eaf2ff;
padding: 16px;
border: 2px solid #0984e3;
margin: 20px;
border-radius: 10px;
width: 300px;
box-sizing: border-box;
}
  • padding: space inside the box
  • border: the edge
  • margin: space outside the box
  • border-radius: rounded corners

Layout: Flex & Grid

display: flex flex-direction justify-content align-items gap display: grid grid-template-columns grid-gap

Flexbox Example

.container {
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
gap: 20px;
}

Grid Example

.grid {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 24px;
}
Flex is best for rows/columns. Grid is best for layouts with rows and columns.

Responsive Design (Media Queries)

@media (max-width: 600px) {
h1 { font-size: 1.4rem; }
.container { padding: 8px; }
}
Use @media rules to make your site look good on all screens (mobile, tablet, desktop).

Position

position: static position: relative position: absolute position: fixed top left z-index
.box {
position: absolute;
top: 50px;
left: 100px;
z-index: 10;
}
  • relative: moves element, keeps its space
  • absolute: places anywhere inside nearest parent
  • fixed: stays in place when scrolling

Transitions & Animation

.button {
background: #0984e3;
color: #fff;
transition: background 0.3s, transform 0.2s;
}
.button:hover {
background: #fdcb6e;
color: #222;
transform: scale(1.08);
}
  • transition: smooth property changes
  • transform: scale, move, rotate
  • animation: for more complex movement (see W3Schools/MDN for keyframes)

Best Practices

  • Keep CSS in a separate file (style.css) or in <style> in <head>.
  • Use classes for reusable styles, ids for unique styles.
  • Comment your code: /* This is a comment */
  • Use rem or em for font sizes for better scaling.
  • Use browser DevTools (F12) to test CSS live.

Mini Projects & Exercises

1. Stylish "About Me" Card
<div class="card">
<h2>webTune1</h2>
<p>I love coding!</p>
</div>

<style>
.card {
background: #fffbe4;

border-radius: 16px;

text-align: center;
width: 280px;
margin: 18px auto;
font-family: 'Figtree', Arial, sans-serif;
}
.card h2 { color: #0984e3; }
</style>
2. Responsive Two-Column Layout
<div class="row">
<div class="col">Left</div>
<div class="col">Right</div>
</div>

<style>
.row { display: flex; gap: 18px; }
.col { flex: 1; background: #eaf2ff; padding: 18px; border-radius: 7px; }
@media (max-width:600px) {
.row { flex-direction: column; }
}
</style>
Exercises:
  • Make a button with rounded corners and a hover effect.
  • Change background color of your page based on the time of day (hint: use JS).
  • Try making a 3-column grid for your favorite foods.
