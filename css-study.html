What is CSS?

CSS means Cascading Style Sheets. It controls the look: color, size, layout, fonts, and more.

Basic CSS Example

<style>
body { background: #f0f8ff; }
h1 { color: #0984e3; font-size: 38px; }
p { color: #222; font-size: 20px; }
</style>
Put <style> in your <head> section.

Box Model: Padding, Border, Margin

.box { background: #eaf2ff; padding: 12px; border: 2px solid #0984e3; margin: 20px; }
Mini Project: Style your About Me page!

CSS for Buttons

button { background: #0984e3; color: #fff; border: none; border-radius: 6px; padding: 7px 22px; cursor: pointer; } button:hover { background: #04507a; }
