What is HTML?

HTML (HyperText Markup Language) is the standard language for building web pages. It tells browsers what to display and how to structure content.

Basic Structure

<!DOCTYPE html>
<html>
<head>
<title>My Page</title>
<meta charset="UTF-8">
</head>
<body>
<h1>Hello, World!</h1>
<p>This is a paragraph.</p>
</body>
</html>
- <!DOCTYPE html>: Declares this is an HTML5 document.
- <html>: Root element.
- <head>: Info for browser (title, meta).
- <body>: Main visible content.

Tip: Not every tag is needed in every project. Learn the basics first!

Common Attributes

  • id: Unique name for an element. <p id="main">
  • class: Group elements for styling. <div class="card">
  • style: Inline CSS. <h1 style="color:red">
  • src: Image or media source. <img src="pic.jpg">
  • href: Link URL. <a href="https://...">
  • alt: Image description for accessibility. <img alt="desc">
  • title: Tooltip text.
  • value: Default value for inputs.
  • type: Input type (text, password, email, etc.).
  • placeholder: Hint text for input.
  • disabled, checked, readonly, required: For forms.

Text & Headings

<h1>Main Title</h1>
<h2>Subtitle</h2>
<p>This is a paragraph.</p>
<strong>Bold</strong> <em>Italic</em>
<br> (line break)   <hr> (horizontal rule)
- Use headings for titles and organize content.
- h1 is biggest; h6 is smallest.
- <br> = new line; <hr> = horizontal line.

Links & Navigation

<a href="https://www.example.com">Visit Example</a>
<a href="page2.html">Go to Page 2</a>
<a href="#section3">Jump to Section 3</a>
- href is the link URL.
- Use #id to jump to a section on the same page.

Lists (ul, ol, li)

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
<ol>
<li>First</li>
<li>Second</li>
</ol>
- ul: Unordered list (bullets).
- ol: Ordered list (numbers).
- li: List item.

Images & Media

<img src="photo.jpg" alt="My Photo" width="200">
<audio controls src="music.mp3"></audio>
<video controls width="300" src="movie.mp4"></video>
- Always use alt for images (for accessibility & SEO).
- controls adds play/pause for audio/video.

Tables

<table>
<thead>
<tr><th>Name</th><th>Age</th></tr>
</thead>
<tbody>
<tr><td>Alice</td><td>21</td></tr>
<tr><td>Bob</td><td>23</td></tr>
</tbody>
</table>
- table: Table wrapper
- tr: Table row
- th: Table header
- td: Table data/cell

Forms (User Input)

<form>
Name: <input type="text" name="name" required><br>
Email: <input type="email" name="email"><br>
Password: <input type="password" name="pass"><br>
Gender:
<input type="radio" name="gender" value="M"> Male
<input type="radio" name="gender" value="F"> Female
Hobbies:
<input type="checkbox" name="hobby" value="music"> Music
<input type="checkbox" name="hobby" value="games"> Games
<button type="submit">Submit</button>
</form>
- type can be text, email, password, checkbox, radio, etc.
- Use required for fields that must be filled.
<select name="city">
<option>Angul</option>
<option>Cuttack</option>
</select>

Semantic HTML (Clean, Accessible Code)

<header>...</header>
<nav>...</nav>
<main>...</main>
<section>...</section>
<article>...</article>
<aside>...</aside>
<footer>...</footer>
- Use these tags to show what each part of your page means.
- Helps with SEO and accessibility (screen readers).

SEO Basics (Get Found on Google!)

  • Always use <title> in <head>.
  • Use <meta name="description"> for page summary.
  • Use headings (h1, h2...) properly.
  • Add alt to all images.
  • Structure with semantic tags.
  • Keep code clean and readable!
<meta name="description" content="Learn HTML A-Z for beginners: tags, forms, SEO, projects.">

Best Practices

  • Always close your tags.
  • Indent code for readability.
  • Use lowercase for tags and attributes.
  • Use alt text for all images.
  • Test your page in different browsers/devices.
  • Comment your code: <!-- This is a comment -->
