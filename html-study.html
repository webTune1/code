What is HTML?

HTML is the language that gives structure to webpages. It uses tags like <h1>, <p>, <a>, <img>, etc.

Basic Structure

<!DOCTYPE html>
<html>
<head>
<title>My Page</title>
</head>
<body>
<h1>Hello!</h1>
<p>This is a paragraph of text.</p>
</body>
</html>
  • <h1>...</h1>: Biggest heading.
  • <p>...</p>: Paragraph.
  • <a href="...">...</a>: Link.
  • <img src="..." alt="...">: Image.
  • <ul><li>: List.
  • <input>, <form>: Forms.

Common Tags

<h1>Heading 1</h1> <h2>Heading 2</h2> <p>Paragraph text here.</p> <a href="https://example.com">Click me</a> <img src="image.jpg" alt="My image"> <ul> <li>First item</li> <li>Second item</li> </ul>

Mini Project: "About Me" Page

<!DOCTYPE html>
<html>
<head>
<title>About Me</title>
</head>
<body>
<h1>My Name is [Your Name]</h1>
<p>I am from Angul, Odisha.</p>
<ul>
<li>Coding</li>
<li>Music</li>
<li>Games</li>
</ul>
<a href="https://www.w3schools.com/">Learn More</a>
</body>
</html>
Try: Change name, add hobbies, add a photo!

HTML Forms

<form>
Name: <input type="text" name="name"><br>
Email: <input type="email" name="email"><br>
<button type="submit">Submit</button>
</form>
