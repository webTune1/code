What is JavaScript?

JavaScript (JS) lets you add interactivity, popups, changes, and more to your web pages.

Basic JS Example

<button onclick="alert('Hello!')">Click me</button>
alert(...) shows a popup message.

Change Text with Button

<h2 id="msg">Welcome!</h2> <button onclick="document.getElementById('msg').innerText = 'You clicked me!';">Click me</button>

Variables & Math

<script> let a = 5; let b = 3; document.write("Sum: " + (a + b)); </script>

Mini Project: Color Changer

<button onclick="document.body.style.backgroundColor = 'yellow';">Yellow</button> <button onclick="document.body.style.backgroundColor = 'lightblue';">Blue</button>
Click the buttons to change the background!

Show/Hide Example

<p id="showme" style="display:none;">Hello, I am webTune1!</p> <button onclick="document.getElementById('showme').style.display='block';">Show Name</button> <button onclick="document.getElementById('showme').style.display='none';">Hide Name</button>

Practice Questions

  • Make a button that increases a number by 1 every time you click.
  • Make an alert that says "Welcome back!" when the page loads.
