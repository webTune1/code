What is JavaScript?

JavaScript (JS) makes web pages interactive: buttons, popups, games, forms, animations â€” it runs in your browser.

Syntax & Output

<script>
alert("Hello!"); // Popup
document.write("Hi there!"); // Write to page
console.log("Debug info"); // Print to browser console
</script>
  • Put JS in <script> tags or a .js file.
  • console.log(...) is your best friend for testing!

Operators

+ - * / % == === != > < >= <= && || !
let a = 5 + 2; // 7
let same = (a === 7); // true
let diff = (a !== 6); // true
let big = (a > 3 && a < 10); // true

If-Else (Conditions)

let score = 65;
if (score >= 50) {
console.log("Pass");
} else {
console.log("Fail");
}
if / else lets you run code based on conditions.

Loops (for, while)

// For loop
for (let i = 1; i <= 5; i++) {
console.log(i);
}

// While loop
let n = 3;
while (n > 0) {
console.log(n);
n--;
}
Loops repeat code; use for when you know how many times, while otherwise.

Functions

function greet(name) {
return "Hello, " + name;
}
let msg = greet("webTune1");
console.log(msg); // "Hello, webTune1"
Functions let you reuse code. Call with greet("Name").

Arrays

let fruits = ["apple", "banana", "mango"];
console.log(fruits[0]); // "apple"
fruits.push("orange"); // add
console.log(fruits.length); // count
Arrays store multiple values. Index starts at 0.

Objects

let user = {
name: "webTune1",
age: 25,
isAdmin: false
};
console.log(user.name); // "webTune1"
Objects group data and functions together.

DOM & Events (Make Pages Interactive)

document.getElementById() innerText addEventListener onclick
<h2 id="msg">Welcome!</h2>
<button onclick="document.getElementById('msg').innerText = 'You clicked!';">Click me</button>
Change text or style with JS. Every element with an id can be accessed in JS.
<button id="alertBtn">Show Alert</button>
<script>
document.getElementById("alertBtn").addEventListener("click", function() {
alert("Button clicked!");
});
</script>
addEventListener is a modern way to handle events.

Mini Projects & Exercises

1. Counter Button
<button onclick="increase()">Increase</button>
<span id="count">0</span>
<script>
let c = 0;
function increase() {
c++;
document.getElementById("count").innerText = c;
}
</script>
2. Change Background Color
<button onclick="document.body.style.backgroundColor = 'pink';">Pink</button>
<button onclick="document.body.style.backgroundColor = 'lightblue';">Blue</button>
3. Simple Calculator
<input id="a" type="number"> + <input id="b" type="number">
<button onclick="sum()">Add</button>
<span id="result"></span>
<script>
function sum() {
let a = Number(document.getElementById("a").value);
let b = Number(document.getElementById("b").value);
document.getElementById("result").innerText = a + b;
}
</script>
Exercises:
  • Make a button that hides and shows your name.
  • Show a random number between 1 and 100 when you click a button.
  • Make a to-do list: add new items to a list when you type and press a button.

Best Practices

  • Use let and const, not var.
  • Always indent code and use clear names.
  • Use console.log() to debug.
  • Keep JS in a separate .js file for real projects.
  • Comment your code: // This is a comment
